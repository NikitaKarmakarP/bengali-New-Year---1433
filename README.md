# 🌸 Bengali New Year Website - 1433

✨ A beautifully designed website celebrating the spirit of **Poila Boishakh (Bengali New Year)** with culture, tradition, and modern web design.

---

## 🚀 Live Demo
🌐 https://bengali-new-year-1433-66v1.vercel.app/

---

## 📌 Project Overview

This project is a tribute to Bengali culture and heritage, showcasing:
- 🎉 Bengali New Year traditions  
- 🍛 Authentic Bengali foods  
- 🎨 Modern and responsive UI design  

The goal of this project is to blend **traditional culture with modern technology** and create an engaging user experience.

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript

✔️ Beautiful and responsive design
